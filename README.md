## Overview

This project is an end-to-end HR analytics case study focused on employee attrition and performance analysis. It combines data cleaning, exploratory data analysis, and interactive dashboard development to uncover insights related to workforce retention, salary distribution, department-level attrition, employee status, and performance segments.
